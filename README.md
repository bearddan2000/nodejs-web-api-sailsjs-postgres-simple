# nodejs-web-api-sailsjs-postgres-simple

## Description
A sailsjs api that queries postgres server.

## Tech stack
- sailsjs
- postgresql

## Docker stack
- node:latest
- postgres:alpine

## To run
`sudo ./install.sh -u`
curl -i localhost/dogs

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
